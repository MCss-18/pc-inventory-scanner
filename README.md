# System Inventory Extractor

A lightweight tool to extract detailed hardware, network, and software information from PCs and laptops. Ideal for system audits, IT inventory, and troubleshooting.

## Features

- CPU, GPU, RAM, storage, motherboard  details 
- Monitors connected to the PC
- OS and version
- Security (BitLocker, secure boot, tpm, UAC)
- Network configuration and status 
- Device brand, model, and serial number (laptops)  
- List of installed applications
- UTC Date audit
- Local users

## Supported Platforms

- Windows (current version)
